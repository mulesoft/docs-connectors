= DocuSign Connector 1.0

Support Category: https://www.mulesoft.com/legal/versioning-back-support-policy#anypoint-connectors[Select]

Anypoint Connector for DocuSign (DocuSign Connector) enables you to access the DocuSign platform and exposes operations provided by the DocuSign eSignature API v2.1. 

For compatibility information and fixed issues, see the DocuSign Connector Release Notes.

include::connectors::partial$prerequisites-partial.adoc[]



== Authentication Types

DocuSign Connector connections use the following authentication methods:

* Authorization Connection Provider (Implicit Grant)
* OAuth Connection Provider
