= About Connectors and Modules (for Mule 4)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

Connectors facilitate integrations with third-party APIs. Modules are extensions to the Mule runtime that you can use when building a Mule application. 

[IMPORTANT]
====
For Mule 3 connectors, see xref:3.9@mule-runtime::anypoint-connectors.adoc[Anypoint Connectors (for Mule 3)].
====
