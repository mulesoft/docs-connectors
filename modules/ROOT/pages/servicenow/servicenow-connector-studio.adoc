To configure a connector in Anypoint Studio, follow this process:

. Install the connector.
. Configure the connector.
. Configure an input source for the connector.

== Install the Connector

. In Studio, click the Anypoint Exchange icon in the Studio task bar.
. Click *Login* in Exchange and supply your Anypoint Platform username and password.
. Search for the ServiceNow connector, click on the connector in Exchange, and click *Add to project*.
. Follow the prompts to install the connector.

== Configure the ServiceNow Connector

. Drag and drop the connector to the Studio Canvas.
. Configure the Global element for the connector:
+
image::servicenow/servicenow-configStudio1.png[servicenow-config-studio]
+
[%header%autowidth.spread]
|===
|Field |Description
|*Username* | Username to log in with.
|*Password* | Corresponding password.
|*Service Address* | WSDL endpoint.
|*ServiceNow Version* | ServiceNow version.
|*User Table List*| Custom tables for the logged-in user.
|===
+
. Go to the Transport Tab and configure a Basic HTTP Transport:
+
// image::<missing>configStudio2.png[config connector 2]
+
[%header%autowidth.spread]
|===
|Field |Description
|*Username*| Enter the username to log in with.
|*Password*| Enter the corresponding password.
|===

Configure an input source for the ServiceNow connector such as an HTTP Listener, or Scheduler.
